# 4phyiscsBodies
+ The User drags the RED Box when touching the screen.
+ 4 SKPhysicsDody's Are Connected by SKPhysicsJoint.
+ SKPhysicsJoint programmed with "rope" behavior.
+ PhysicsWorld's gravity set to default.
